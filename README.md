# Siyuan plugin background grid

可设置形状和颜色的背景纹路插件，支持配置网格与点阵背景，支持配置颜色、大小、间距

## 使用方法

应用插件后，右上角按钮中找到“背景纹路”，点击出现菜单，选择设置即可进行配置；

+ 类型：支持“方格”和“点阵”
+ 间距：配置方格或点阵的间距
+ 大小：配置线的宽度或点的大小
+ 颜色：配置方格和点阵的颜色
+ 透明度：配置颜色的透明值，1-100，100为不透明
